What's the Difference

How OrderNChaos differs from Nirvana

Previous: Introduction Next: Quick Start

Last updated 3 years ago

While OrderNChaos (ONC) follows the core concepts of Nirvana, there are several key differences and enhancements in the ONC protocol:

Differences from Nirvana

  • Cross-Chain Compatibility: Unlike Nirvana which is built on Solana, ONC is implemented in Solidity and can be deployed on any EVM-compatible blockchain. The initial launch is planned for the Avalanche network.
  • Enhanced Staking Mechanics: ONC introduces improved staking mechanics with the $prCHAOS token, which offers more sustainable yield generation for stakers.
  • Improved Price Discovery: The vAMM (virtual Automated Market Maker) in ONC offers enhanced price discovery mechanisms to ensure more stable and predictable price movements.
  • Expanded Reserve Assets: ONC maintains a more diverse basket of reserve assets, providing better stability and resilience to market fluctuations.

Key Token Properties

$CHAOS

The $CHAOS token is an algorithmic metastable token that serves as a store of wealth with the following properties:

  • Backed by a diverse basket of reserve assets
  • Has a hard floor price that cannot be breached
  • Benefits from a rising floor price mechanism
  • Can be staked to earn $prCHAOS rewards
  • Can be used as collateral for $ORDER loans

$ORDER

The $ORDER token is a superstable store of value with the following properties:

  • Fully collateralized 1:1 with the intrinsic value of $CHAOS
  • Zero liquidation risk for loans
  • Negative interest rate (you earn yield on your debt)
  • Concentrates liquidity from various stablecoins

These differences and enhancements make OrderNChaos a more versatile and robust algorithmic stablecoin system compared to its predecessors.
