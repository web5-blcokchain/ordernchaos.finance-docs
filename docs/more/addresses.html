How to Verify Contract Addresses

To ensure you are interacting with the official OrderNChaos protocol contracts, follow these verification steps:

  1. Official Sources: Always check addresses against multiple official sources, including:
    • The official OrderNChaos website (https://ordernchaos.finance)
    • This documentation page
    • Official social media accounts
    • Official Discord or Telegram announcements
  2. Block Explorers: Verify contract details on block explorers like Etherscan, Arbiscan, or Optimism Explorer:
    • Check if the contract is verified
    • Look for the official deployment transaction
    • Check contract creation date (should match announced launch dates)
  3. Smart Contract Verification: For technically-inclined users, verify the contract source code matches the expected functionality

Add to Wallet

For convenience, you can add these tokens to your wallet directly from the OrderNChaos app. This ensures you're using the correct contract addresses. Navigate to the "Connect Wallet" section and look for the "Add Tokens" option.

Contract Audits

All OrderNChaos smart contracts have been audited by multiple security firms to ensure safety and reliability. You can find the full audit reports on the Security page of the official website.

Main auditors:

  • CertiK - Comprehensive audit of all core contracts
  • Hacken - Secondary audit with focus on economic mechanisms
  • PeckShield - Specialized audit of the Zero Risk Loan system
